> 最近接触到eda相关的领域，设计到eda与agent结合的技术，于是总结了一下eda相关的一些术语里面各个字母的含义

## <span id="EDA_1"></span>关于EDA的各个术语

### <span id="_2"></span>流程与领域

- **EDA** = **E**lectronic **D**esign **A**utomation（电子设计自动化）
- **IC** = **I**ntegrated **C**ircuit（集成电路）
- **ASIC** = **A**pplication-**S**pecific **I**ntegrated **C**ircuit（专用集成电路）
- **SoC** = **S**ystem on **C**hip（片上系统）

------------------------------------------------------------------------

### <span id="_11"></span>设计描述相关

- **HDL** = **H**ardware **D**escription **L**anguage（硬件描述语言）
- **RTL** = **R**egister-**T**ransfer **L**evel（寄存器传输级）
- **Verilog**：不是字母缩写（语言名字）
- **SV**（常写）= **S**ystem**V**erilog（SystemVerilog 的简称；严格说是缩写用法）
- **VHDL** = **V**HSIC **H**ardware **D**escription **L**anguage  
  其中 **VHSIC** = **V**ery **H**igh **S**peed **I**ntegrated **C**ircuit（超高速集成电路，国防项目名）

------------------------------------------------------------------------

### <span id="_22"></span>约束/脚本/分析

- **Tcl** = **T**ool **C**ommand **L**anguage（工具命令语言）
- **SDC** = **S**ynopsys **D**esign **C**onstraints（Synopsys 设计约束）
- **STA** = **S**tatic **T**iming **A**nalysis（静态时序分析）
- **WNS** = **W**orst **N**egative **S**lack（最坏负裕量）
- **TNS** = **T**otal **N**egative **S**lack（总负裕量）
- **SDF** = **S**tandard **D**elay **F**ormat（标准延迟格式）

------------------------------------------------------------------------

### <span id="_33"></span>物理设计（后端）

- **P&R** = **P**lace **&** **R**oute（布局与布线）
- **CTS** = **C**lock **T**ree **S**ynthesis（时钟树综合）
- **DEF** = **D**esign **E**xchange **F**ormat（设计交换格式）
- **LEF** = **L**ibrary **E**xchange **F**ormat（库交换格式）
- **GDSII**：传统叫法/行业术语（常被解释为 *Graphic Data System II*，即 **G**raphic **D**ata **S**ystem **II**；“II”表示第二代）
- **OASIS** = **O**pen **A**rtwork **S**ystem **I**nterchange **S**tandard（开放版图系统交换标准）

------------------------------------------------------------------------

### <span id="_44"></span>版图/制造相关检查

- **DRC** = **D**esign **R**ule **C**heck(ing)（设计规则检查）
- **LVS** = **L**ayout **V**ersus **S**chematic（版图对原理图/网表一致性）
- **PDK** = **P**rocess **D**esign **K**it（工艺设计套件）
- **PDN** = **P**ower **D**elivery **N**etwork（供电网络/电源分配网络）

------------------------------------------------------------------------

### <span id="_53"></span>寄生、电源与可靠性

- **SPEF** = **S**tandard **P**arasitic **E**xchange **F**ormat（标准寄生交换格式）
- **RC**：不是严格缩写，电学里 **R**=Resistance（电阻），**C**=Capacitance（电容）
- **IR drop**：**I**=Current（电流），**R**=Resistance（电阻），指电源压降
- **EM** = **E**lectro**m**igration（电迁移）

------------------------------------------------------------------------

### <span id="_62"></span>指标与接口常见缩写

- **PPA** = **P**erformance / **P**ower / **A**rea（性能/功耗/面积）
- **I/O** = **I**nput / **O**utput（输入/输出）
